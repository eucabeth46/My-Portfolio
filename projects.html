Mama Guardian AI

The "Mama Guardian AI" project addresses Kenya's high maternal mortality rates by developing a predictive health analytics tool. This system uses supervised machine learning to analyze maternal health data and identify high-risk pregnancies early. The platform provides risk assessments to healthcare providers, facilitates timely interventions, and supports Kenya's Universal Health Coverage goals. It includes a mobile app for data collection, a web dashboard for monitoring, and SMS alerts for at-risk mothers and community health workers.

Technologies: HTML, CSS, JavaScript, MongoDB, Flask API

Pharma Check

.Pharma Check proposes an AI-powered solution that leverages computer vision and Optical Character Recognition (OCR) to detect fake drugs through analysis of packaging and pill images. The system will be accessible via a mobile or web application, enabling health workers, pharmacists, and consumers to verify drug authenticity in real time. Using convolutional neural networks (CNNs) and OCR tools like Tesseract, the solution aims to improve medication safety, support regulatory enforcement, and build public trust in pharmaceuticals. The project aligns with multiple UN Sustainable Development Goals, including Good Health and Well-being (SDG 3) and Industry, Innovation and Infrastructure (SDG 9), and is tailored for scalability and local relevance within the Kenyan healthcare context. Analysis of Medication Handling Time in Kenyatta National hospital

This project analyzed timestamp data from 91 patient records to assess delays in the hospital medication process across three stages: doctor’s prescription, nurse’s request, and pharmacy issuance. The study revealed median delays of 40 minutes (doctor to nurse), 30 minutes (nurse to pharmacy), and 80 minutes overall, with significant variability and outliers. Data quality issues were identified, including 11.3% missing records and potential negative time differences due to documentation errors or system desynchronization. Key visualizations, such as boxplots, were used to highlight bottlenecks and inconsistencies. The findings emphasized the need for process improvements. Recommendations included digitizing treatment sheets, automating alerts, setting performance benchmarks, and enhancing staff training to ensure timely and safe medication delivery.

Tools used: Data collection using excel spreadsheets.
Data analysis (cleaning and preprocessing using Python Pandas,NumPy,Matplotlib and Seaborn libraries).
Data visualization ( used histogram and boxplots).

Impact: Improved understanding of workflow inefficiencies and highlighted critical areas for intervention to enhance patient care and data reliability in medication handling.
